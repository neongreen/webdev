# Как подключить Ant Design

Ant Design ([https://ant.design](https://ant.design)) — это популярная библиотека компонентов для React.

## Если проект на Next.js

Открыть терминал с помощью <kbd>Ctrl+`</kbd> (VSCode).

В терминале выполнить:

```bash Terminal
npm i -S antd
```
