# `Form.useWatch`

С помощью `Form.useWatch` можно создать переменную, которая будет автоматически обновляться, когда пользователь вводит данные в форму.

Для того, чтобы использовать `Form.useWatch`, надо сначала создать ссылку на форму с помощью `Form.useForm`:

```jsx page.js
export default function Page() {
  const [form] = Form.useForm()
  const email = Form.useWatch('email', form)
  return (
    <Form form={form}>
      <Form.Item name="email">
        <Input />
      </Form.Item>
      ...
    </Form>
  )
}
```

Ниже есть пример формы с именем, фамилией, отчеством, и галочкой "Нет отчества".

* Переменная `noPatronymic` следит за галочкой "Нет отчества".

* У поля "Отчество" есть атрибут `hidden={noPatronymic}`. Это значит, что поле будет скрыто, если в переменной `true` — то есть, если галочка "Нет отчества" отмечена.
