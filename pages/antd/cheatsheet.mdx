## Кнопки ([официальная справка](https://ant.design/components/button/)) <a id="buttons"></a>

Импорты для всех примеров ниже:

```jsx
import { Button } from 'antd'
```

Ключевые атрибуты:

* `type` — тип кнопки: `"primary"`, `"dashed"`, `"text"`, `"link"`
* `size` — размер кнопки: `"small"`, `"large"`
* `shape` — форма кнопки: `"circle"`, `"round"`
* `danger` — красная кнопка
* `disabled` — отключенная кнопка
* `loading` — кнопка с индикатором загрузки
* `icon` — иконка слева от текста
* `block` — кнопка занимает всю ширину родителя

События:

* `onClick` — вызывается по нажатию на кнопку
