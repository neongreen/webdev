## Шаблон

Вот шаблон для решения задач. Надо выбирать "Node.js" (а не "JavaScript") в выпадающем списке языков.

```js
let __input = "", __line = 0, readline = () => __input[__line++]
process.stdin.resume().setEncoding("utf-8")
    .on("data", s => __input += s)
    .on("end", _ => { __input = __input.trim().split("\n").map(s => s.trim()); main() })

function main() {
    // Решение писать здесь
}
```
