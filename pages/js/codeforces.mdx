## Шаблон

Вот шаблон для решения задач. Надо выбирать "Node.js" (а не "JavaScript") в выпадающем списке языков.

```js
process.stdin.resume()
process.stdin.setEncoding("utf-8")
let __input = "", __line = 0
process.stdin.on("data", s => __input += s)
process.stdin.on("end", _ => {
    __input = __input.trim().split("\n").map(s => s.trim())
    main()
})
function readline() { return __input[__line++] }

function main() {
    // Решение писать здесь.
    // 
    // * Функция `readline()` возвращает одну строку из ввода.
    // * Выводить можно с помощью `console.log()`.
}
```

## Как читать ввод

<Boxes>
  <Box>
    **Строчка с одним числом,** например:
    * `5`
  </Box>
  <Box>
    <CodeAnn ok="n = 5">
      ```js
      let n = Number(readline())
      ```
    </CodeAnn>
  </Box>
</Boxes>

<Boxes>
  <Box>
    **Строчка с тремя числами,** например:
    * `3 5 8`
  </Box>
  <Box>
    <CodeAnn ok="n = 3, m = 5, k = 8">
      ```js
      let [n, m, k] = readline().split(" ").map(Number)
      ```
    </CodeAnn>
  </Box>
</Boxes>

<Boxes>
  <Box>
    **Строчка с каким-то произвольным количеством чисел,** например:
    * `3 5 8 13 21`
  </Box>
  <Box>
    <CodeAnn ok="xs = [3, 5, 8, 13, 21]">
      ```js
      let xs = readline().split(" ").map(Number)
      ```
    </CodeAnn>
  </Box>
</Boxes>

<Boxes>
  <Box>

  </Box>
