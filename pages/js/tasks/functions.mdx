# Задания на функции

Напишите функции по заданию. (Первая функция уже написана.)

1.  **Функция <code>max</code> должна возвращать наибольшее из двух чисел:**

    <FunctionChecker
      taskId="max"
      expected={([a,b]) => Math.max(a,b)}
      args={[{from: 0, to: 5}, {from: 0, to: 5}]}
      code={endent`
        function max(a, b) {
          if (a > b) {
            return a
          } else {
            return b
          }
        }
      `}
    />

1.  **Функция `factorial(n)` должна перемножать числа от 1 до `n`.** Например, `factorial(4)` должна возвращать 24, потому что <InlineMath math="1 \times 2 \times 3 \times 4 = 24" />.

    <FunctionChecker 
      taskId="factorial"
      expected={([n]) => {
        let result = 1
        for (let i = 1; i <= n; i++) result *= i;
        return result
      }}
      args={[{from: 1, to: 10}]}
    />

1.  **Функция `isLeap(year)` должна проверять, високосный год или нет.** (Високосный год — это такой, в котором есть лишний день.) Если год високосный — возвращать `true`, иначе `false`.

    Год считается високосным, если он делится на 4, но не делится на 100. Однако, если год делится на 400, то он также является високосным. Например, 2000 год является високосным, но 1900 и 2100 года — нет.

    <FunctionChecker
      taskId="isLeap"
      expected={([year]) => year % 400 === 0 || (year % 4 === 0 && year % 100 !== 0)}
      args={[{from: 1900, to: 2400}]}
    />
