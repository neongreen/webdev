# Задания на циклы

<style jsx>{`
  .list code {
    margin-left: 20px;

  `}</style>

Напоминаю синтаксис для циклов `for`:

```js
// for (начало; условие; шаг) { тело цикла }

for (let i = 1; i <= 10; i++) {
  console.log(i)
}
```

Например, цикл выше начнется с `i = 0` и будет продолжаться, пока `i <= 10`. После каждой итерации цикла `i` будет увеличиваться на `1`. То есть, цикл выведет числа от 1 до 10 и остановится.

## Штриховка

_Эти задания надо делать в [editor.p5js.org](https://editor.p5js.org/)._

С помощью одного цикла `for` нарисуйте такую штриховку:

<Boxes>
  <Box>
    ### 1
    <P5 sketch={(p5) => {
      p5.setup = () => p5.createCanvas(200, 200, p5.P2D)
      p5.draw = () => {
        p5.background(220)
        for (let i = 1; i < 20; i++) p5.line(10, i * 10, 190, i * 10)
      }
    }} />
  </Box>
  
  <Box>
    ### 2
    <P5 sketch={(p5) => {
      p5.setup = () => p5.createCanvas(200, 200, p5.P2D)
      p5.draw = () => {
        p5.background(220)
        for (let i = 1; i < 20; i++) p5.line(i * 10, 10, i * 10, 190)
      }
    }} />
  </Box>

  <Box>
    ### 3
    <P5 sketch={(p5) => {
      p5.setup = () => p5.createCanvas(200, 200, p5.P2D)
      p5.draw = () => {
        p5.background(220)
        for (let i = 1; i < 20; i++) {
          p5.line(10, i*10, 190, i*10)
          p5.line(i*10, 10, i*10, 190)
        }
      }
    }} />
  </Box>
</Boxes>
