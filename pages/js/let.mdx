# JavaScript: `let`

<Ann text="пусть">`let`</Ann> создает _переменную_.

Переменная — это коробочка, в которой может лежать какое-то значение.

```js .js
let название = значение;
```

Значения переменных можно менять с помощью `=` (читается "присваивание").

```js .js
переменная = новое значение;
```
