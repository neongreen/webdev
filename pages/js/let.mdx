# JavaScript: `let`

<Ann text="пусть">`let`</Ann> создает _переменную_.

Переменная — это коробочка, в которой может лежать какое-то значение.

Значения переменных можно менять с помощью знака `=` (читается "присваивание").

<Boxes>
  <Box>
    **Создаем новую переменную `x`:**

    ```js JavaScript
    let x = 0;
    ```
  </Box>
  <Box>
    **Меняем значение `x`:**

    ```js JavaScript
    x = 1;
    ```
  </Box>
</Boxes>

Попробуем пример.

## `n++` (инкремент)

Увеличивать переменные на единицу приходится часто, поэтому вместо `n = n + 1` можно писать просто `n++`.

<Exercise>
  <Step>
    Поменяйте `n = n + 1` в примере выше на `n++` и проверьте, что он работает точно так же.
  </Step>
  <Mistake>
    **Проверьте, что `n++` набрано без пробелов.** `n + +` — не сработает.
  </Mistake>
</Exercise>

## `n--` (декремент)

<Exercise>
  <Step>
    Поменяйте `n++` в примере выше на `n--`. Теперь число должно уменьшаться, а не увеличиваться.
  </Step>
</Exercise>
