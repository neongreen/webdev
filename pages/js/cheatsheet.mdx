## `if` и условия <a name="if" />

Сделать что-то, только **если** выполняется условие:

```js
if (условие) { 
  действие
}
```

Если выполняется условие, сделать одно, а **иначе** что-то другое:

```js
if (условие) { 
  действие
} else {
  другое действие
}
```

Цепочка условий:

```js
if (...) { 
  действие
} else if (...) {
  другое действие
} else {
  третье действие
}
```
