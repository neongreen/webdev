# JavaScript: `if`

<Ann text="если">`if`</Ann> выполняет кусочек кода, только **если** верно какое-то условие.

<Exercise>
  <Step>
    Попробуем простой пример с `if`. Например, у нас есть кнопка и мы хотим писать пользователю "Хватит", если нажать на кнопку больше трех раз.

    Для этого надо а) считать, сколько раз нажали на кнопку и б) проверять, больше трех раз или нет. Вот пример, который делает обе эти вещи. Перепечатайте его и проверьте, что он работает.

    <Boxes>
      <Box>
        ```html index.html
        <button onclick="f()">
          Нажми меня
        </button>
        ```
      </Box>
      <Box>
        ```js script.js mark=5:7
        let count = 0;

        function f() {
          count++;
          if (count > 3) {
            alert("Хватит");
          }
        }
        ```
      </Box>
    </Boxes>
  </Step>
  <Mistake>
    **Скобки вокруг условия обязательны.** Нельзя писать `if count > 3`. В других языках бывает можно, а в JavaScript нельзя.
  </Mistake>
  <Mistake>
    **В `onclick` после названия функции тоже нужны скобки.** Если написать `onclick="f"`, функция просто не будет вызываться.
  </Mistake>
</Exercise>

Еще полезно уметь сделать "что-то одно, а иначе — что-то другое". Для этого есть слово <Ann text="иначе">`else`</Ann>, которое используется вот так:

```js
if (условие) {
  // выполнится, если условие верно
} else {
  // выполнится, если условие не верно
}
```

<Exercise>
  <Step>
    Добавьте `else` в предыдущий пример, чтобы, если кнопка нажата три раза или меньше, выдавалось сообщение "Давай еще".
  </Step>
</Exercise>

Иногда хочется сделать цепочку `if`. Например:

* Если `count > 3`, выдай вот такое сообщение,
* а если `count === 3`, то вот такое,
* (...тут могут быть еще какие-то условия,)
* а если все оказалось неверно, тогда сделай вот то-то.

Тогда надо писать так:

```js
if (условие 1) {
  // штука 1
} else if (условие 2) {
  // штука 2
} else if (условие 3) {
  // штука 3 (может быть еще сколько угодно)
} else {
  // последняя штука
}
```

<Exercise>
  <Step>
    Поменяйте предыдущий пример еще раз, чтобы:

    * если кнопка нажата меньше трех раз, было "Давай еще",
    * если кнопка нажата три раза, было "Норм",
    * а иначе было "Хватит".
  </Step>
  <Mistake>
    Просто `else (условие)` не сработает, надо именно `else if`.
  </Mistake>
  <Mistake>
    В отличие от (например) Питона, `elif` писать нельзя.
  </Mistake>
  <Mistake>
    В данном конкретном случае можно обойтись просто тремя `if` — и не использовать `else` или `else if` вообще. Но я хочу с `else` и `else if`, потому что это упражнение.
  </Mistake>
</Exercise>

Если надо проверить два условия одновременно — например, "кнопка нажата больше пяти раз и меньше десяти" — надо использовать *булевы операторы*. Про них написано в другом уроке.
