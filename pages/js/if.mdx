# JavaScript: `if`

## <code>if ... <Ann text="иначе">else</Ann> ...</code>

Иногда хочется сделать **что-то одно, если условие верно, и что-то другое — если неверно.**

Для этого есть слово <Ann text="иначе">`else`</Ann>. Его можно использовать только вместе с `if`.

```js
if (условие) {
  действие
} else {
  другое действие
}
```

<Exercise>
  <Step>
    Добавьте `else` в предыдущий пример, чтобы, если кнопка нажата три раза или меньше, выдавалось сообщение "Давай еще". Если кнопка нажата больше трех раз, то все еще должно быть сообщение "Хватит".
  </Step>
  <Mistake>
    **После `else` нельзя писать еще какое-то условие.** `else` значит "если прошлое условие не выполнилось".
  </Mistake>
</Exercise>

## `else if`

Иногда хочется сделать много условий. Например:

* Если `count > 3`, выдай вот такое сообщение,
* а если `count === 3`, то вот такое,
* иначе вот такое.

Тогда надо писать так:

```js
if (условие 1) {
  действие
} else if (условие 2) {
  другое действие
} else if (условие 3) {
  третье действие
} else {
  действие, если все остальное не сработало
}
```

<Exercise>
  <Step>
    Поменяйте предыдущий пример еще раз, чтобы:

    * если кнопка нажата **меньше трех раз,** было "Давай еще",
    * если кнопка нажата **ровно три раза,** было "Норм",
    * а иначе было "Хватит".

    Используйте `else if`.
  </Step>
  <Mistake>
    В отличие от (например) Питона, `elif` писать нельзя.
  </Mistake>
</Exercise>

Если надо проверить два условия одновременно — например, "кнопка нажата больше пяти раз и меньше десяти" — надо использовать *булевы операторы*. Про них написано в другом уроке.
