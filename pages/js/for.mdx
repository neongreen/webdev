# JavaScript: `for`

<Ann text="для">`for`</Ann> — это _цикл_. Он дает сделать что-то несколько раз.

В общей форме цикл выглядит так:

```js .js
for (начало; условие; шаг) {
  действие
}
```

Общая форма непонятная, поэтому смотрим конкретный пример.

  </Step>
</Exercise>

Цикл из примера выше работает так:

* Создаем новую переменную `i`, в ней лежит `0`.
* `i = 0`; выполняем действие, увеличиваем `i` на единицу.
* `i = 1`; выполняем действие, увеличиваем `i` на единицу.
* `i = 2`; выполняем действие, увеличиваем `i` на единицу.
* `i = 3`; **условие больше не выполняется, цикл закончился.**
