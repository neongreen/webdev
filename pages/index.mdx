# Веб-разработка

Курс по веб-разработке, на русском, с нуля. Автор — [Артём Казак](https://artyom.me).

**Темы —** сначала HTML, CSS, JS. Потом фуллстек с [TypeScript](https://www.typescriptlang.org/) + React ([Next.js](https://nextjs.org/)) + [Ant Design](https://ant.design/) + [MongoDB](https://www.mongodb.com/).

**Цель курса —** научиться своими руками что-то делать. Или хотя бы в будущем, когда захочется делать, чтобы можно было украсть у себя прошлого.
