## JSX <a name="jsx" />

JSX — это язык, на котором мы пишем страницы. Он выглядит как смесь JavaScript и HTML.

## Отличия React от HTML <a name="differences" />

Отличия в атрибутах:

<B.Table bordered striped>
  <thead>
    <tr>
      <th>Отличие</th>
      <th>HTML</th>
      <th>React</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Атрибут `class` называется `className`</td>
      <td>
        ```html mark=1[9:17]
        <button class="x">Привет</button>
        ```
      </td>
      <td>
        ```jsx mark=1[9:21]
        <button className="x">Привет</button>
        ```
      </td>
    </tr>

    <tr>
      <td>Атрибут `style` должен содержать объект</td>
      <td>
        ```html mark=1[9:37]
        <button style="background-color: red">
          Привет
        </button>
        ```
      </td>
      <td>
        ```jsx mark=1[9:40]
        <button style={{backgroundColor: 'red'}}>
          Привет
        </button>
        ```
      </td>
    </tr>

    <tr>
      <td>Атрибуты обработчиков событий (`onclick` и так далее) пишутся с большими буквами</td>
      <td>
        ```html mark=1[9:15]
        <button onclick="f()">Привет</button>
        ```
      </td>
      <td>
        ```jsx mark=1[9:15]
        <button onClick={f}>Привет</button>
        ```
      </td>
    </tr>

      <td>Атрибут `oninput` называется `onChange`</td>
