# React — шпаргалка

## JSX

JSX — это язык, на котором мы пишем страницы. Он выглядит как смесь JavaScript и HTML.

        Простейший компонент
      </td>
      <td>
        Компонент — это маленький кусочек, который можно использовать внутри других компонентов.
        
        Из компонентов строится страница.

        Названия компонентов обязательно начинаются с большой буквы.

        ```jsx
        function Foo() {
          return (
            <div>Привет</div>
          );
        }
        ```
      </td>
    </tr>

    <tr>
      <td>
        Фрагменты
      </td>
      <td>
        Если компонент состоит из нескольких элементов, их надо обернуть в `<> </>` ("фрагмент").

        ```jsx
        function Foo() {
          return (
            // mark
            <>
              <div>Привет</div>
              <div>Как дела</div>
            // mark
            </>
          );
        }
        ```
      </td>
    </tr>

    <tr>
      <td>
        Использование компонентов
      </td>
      <td>
        Если компонент называется `Foo`, внутри другого компонента можно просто написать <span style={{whiteSpace: "nowrap"}}>`<Foo />`</span>.

        ```jsx
        function Bar() {
          return (
            <>
              <h1>Заголовок</h1>
            // mark
              <Foo />
            </>
          );
        }
        ```
      </td>
    </tr>
    
    <tr>
      <td>
        Вложенный JS
      </td>
      <td>
        Чтобы вставить в компонент JS-код, его надо обернуть в фигурные скобки.

        ```jsx mark=4[13:15]
        function Foo() {
          let n = 0;
          return (
            <button>{n}</button>
          );
        }
        ```

        Внутри фигурных скобок может быть любое выражение.

        ```jsx mark=3[13:21]
        function Foo() {
          return (
            <button>{ 2 + 2 }</button>
          );
        }
        ```
      </td>
    </tr>

    <tr>
      <td>
        Атрибуты
      </td>
      <td>
        Атрибуты задаются так же, как и в HTML.

        ```jsx mark=3[10:22],3[24:32]
        function Foo() {
          return (
            <img width="300px" src="..." />
          );
        }
        ```

        Если атрибут — не строка, то он обязан быть в фигурных скобках.

        ```jsx mark=3[13:39]
        function Foo() {
          return (
            <button onClick={() => alert("Hi")}>Привет</button>
          );
        }
        ```
      </td>
    </tr>
  </tbody>
