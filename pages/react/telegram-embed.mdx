# Как вставить пост из телеграм-канала на сайт

Будем использовать библиотеку [react-telegram-embed](https://www.npmjs.com/package/react-telegram-embed).

Сначала надо добавить библиотеку в зависимости:

```bash Terminal
npm i -S react-telegram-embed
```

Следующий шаг зависит от фреймворка.

## Next.js

Создать файл <FileName>components/TelegramPost.js</FileName>:

```jsx components/TelegramPost.js
import dynamic from 'next/dynamic'

// @ts-ignore
export const TelegramPost = dynamic(() => import('react-telegram-embed'), {
  ssr: false,
})
```
