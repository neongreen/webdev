# Как создать новый проект с Next.js

Next.js ([nextjs.org](https://nextjs.org)) — это библиотека, чтобы делать сайты и веб-приложения.

Здесь написана инструкция, как сделать проект на Next.js.

## Установка Visual Studio Code

<Boxes>
  <Box>
    Visual Studio Code — это такой клевый редактор.

    Гуглите "Visual Studio Code" и делаете по инструкции.
  </Box>
  <Box>
    <Figure image="/vsc.png" noBorder />
  </Box>
</Boxes>

## Установка Node.js

<Boxes>
  <Box>
    Node.js — это штука, которая умеет запускать серверый код на JavaScript. То есть, не код, который рисует сайт, а код, который отвечает на запросы, общается с базой данных и так далее.

    Открываете [nodejs.org](https://nodejs.org/) и устанавливаете. Лучше брать версию, у которой рядом буквы "LTS".

    На все вопросы установщика можно отвечать "да" или "next".
  </Box>
  <Box>
    <Figure image="/nodejs.png" />
  </Box>
</Boxes>

## Создание проекта

1. **Создаете папку в любом месте.** В этой папке будет проект.

   В названии папки не должно быть пробелов, кириллицы, вроде бы большие буквы тоже могут как-то помешать. Можно назвать `learn`, например. Или `site`.

2. <Boxes>
     <Box>
       **В Visual Studio Code ищете пункт "Open folder" в меню "File" и выбираете эту папку.**

       Если Visual Studio Code на русском — то "Открыть папку" в меню "Файл".
     </Box>
     <Box>
       <Figure image="/open-folder.png" />
     </Box>
   </Boxes>

3. **Открываете терминал в Visual Studio Code.** (Сочетание клавиш <kbd>Ctrl+`</kbd>.)

4.  <Boxes>
      <Box>
        **Вводите команду:**

        ```bash Terminal
        npx create-next-app .
        ```
 
        Энтер. Потом еще раз энтер.
 
        Ждете, пока оно все установит. Если почему-то прервалось — ту же команду заново.
      </Box>
      <Box>
        <Figure image="/npx-create-react-app.png" />
      </Box>
    </Boxes>

<B.Alert variant="warning">
  Если вы видите ошибку, что <code>npx</code> не найден, то надо закрыть Visual Studio Code и открыть его снова.
  
  Если все еще не работает — значит, Node.js не установился или установился криво.
</B.Alert>

## Запуск проекта

В терминале:

```bash Terminal
npm run dev
```

И открыть в браузере адрес [localhost:3000](http://localhost:3000).
