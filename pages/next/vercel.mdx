# Деплой на Vercel

"Деплой" — это когда выкладываешь сайт куда-то в интернет и он там работает (а не локально на компьютере).

Деплоить мы будем с помощью Vercel ([vercel.com](https://vercel.com)), потому что он бесплатный и хорошо работает. Но вообще есть тысяча разных других штук, которые можно использовать.

## Регистрация

Пойти на [vercel.com](https://vercel.com) и там зарегистрироваться.

## Установка команды `vercel`

Теперь надо добавить в проект команду `vercel`, с помощью которой мы будем деплоить:

```bash Terminal
npm install -D vercel
```

И еще нужно залогиниться — только один раз, в дальнейшем не придется:

```bash Terminal
npx vercel login
```

## Ручной деплой

Чтобы задеплоить сайт, нужна вот такая команда:

```bash Terminal
npx vercel -c --prod
```

В отличие от глитча, здесь надо деплоить каждый раз, когда делаете какие-то изменения локально.

## Как поменять адрес сайта

Адрес сайта можно поменять на vercel.com в настройках проекта.

<Boxes>
  <Box>
    Сначала надо зайти на [vercel.com/dashboard](https://vercel.com/dashboard) и там будет список проектов. Надо нажать на только что созданный проект.
  </Box>
  <Box>
    <Figure image="/vercel-dashboard.png" />
  </Box>
</Boxes>

<Boxes>
  <Box>
    Теперь надо нажать на "Settings". Там зайти в раздел "Domains" и выбрать домен, помеченный плашкой "Production". Нажать на кнопку "Edit".
  </Box>
  <Box>
    <Figure image="/vercel-project-edit.png" />
  </Box>
</Boxes>

И там выбрать любой домен, который заканчивается на `.vercel.app`.
