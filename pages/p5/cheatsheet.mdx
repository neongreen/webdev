# p5 — шпаргалка

Полный официальный справочник на английском: [p5js.org/reference/](https://p5js.org/reference/)

## Структура программы

Обязательные функции:

<B.Table bordered striped>
  <thead>
    <tr>
      <th>Функция</th>
      <th>Описание</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td><code>function **setup**() {"{...}"}</code></td>
      <td>Эта функция вызовется один раз при запуске программы.</td>
    </tr>
    <tr>
      <td><code>function **draw**() {"{...}"}</code></td>
      <td>Эта функция вызывается в бесконечном цикле. Она должна рисовать **один** кадр.</td>
    </tr>
  </tbody>
</B.Table>

Дополнительные функции (см. полный список в [Events](https://p5js.org/reference/#group-Events) в официальной справке):

<B.Table bordered striped>
  <thead>
    <tr>
      <th>Функция</th>
      <th>Описание</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td><code>function **mousePressed**() {"{...}"}</code></td>
      <td>Вызывается, когда пользователь нажимает на мышку.</td>
    </tr>
    <tr>
      <td><code>function **mouseReleased**() {"{...}"}</code></td>
      <td>Вызывается, когда пользователь отпускает мышку.</td>
    </tr>
    <tr>
      <td><code>function **mouseClicked**() {"{...}"}</code></td>
      <td>Вызывается, когда пользователь кликает мышкой (нажимает и отпускает).</td>
    </tr>
    <tr>
      <td><code>function **keyPressed**() {"{...}"}</code></td>
      <td>Вызывается, когда пользователь нажимает на клавишу клавиатуры.</td>
    </tr>
    <tr>
      <td><code>function **keyReleased**() {"{...}"}</code></td>
      <td>Вызывается, когда пользователь отпускает клавишу.</td>
    </tr>
  </tbody>
</B.Table>

## Заливка

<B.Table bordered striped>
  <thead>

## Текст

<B.Table bordered striped>
  <style jsx>{`
    td:nth-child(1) {

  `}</style>
  <thead>
    <tr>
      <th>Функция</th>
      <th>Описание</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td><code>**text**(str, x, y)</code></td>
      <td>Выводит строку `str` с левым _нижним_ углом в точке `(x, y)`</td>
    </tr>
    <tr>
      <td><code>**textSize**(n)</code></td>
      <td>Меняет размер текста</td>
    </tr>
    <tr>
      <td><code>**textAlign**(horiz, vert)</code></td>
      <td>
        Меняет выравнивание текста (текст будет выводиться не относительно левого нижнего угла, а какого-то другого)
        
        * Варианты для параметра `horiz`: `LEFT`, `CENTER`, `RIGHT`.
        * Варианты для параметра `vert`: `BOTTOM`, `CENTER`, `TOP`, `BASELINE`.
      </td>
    </tr>
    <tr>
      <td><code>**textStyle**(style)</code></td>
      <td>
        Меняет стиль. Варианты: `NORMAL`, `BOLD`, `ITALIC`, `BOLDITALIC`.
      </td>
    </tr>
    <tr>
      <td><code>**textFont**(font)</code></td>
      <td>
        Меняет шрифт. Варианты: `"Georgia"`, `"Arial"`, `"Impact"` и еще много.

        Полный список шрифтов есть в приложении "Font Book" (macOS) или в папке "Шрифты" в панели управления (Windows).
      </td>
    </tr>
  </tbody>
</B.Table>

## Клавиатура

<B.Table bordered striped>
  <style jsx>{`
    td:nth-child(1) {
      white-space: nowrap;
    }
  `}</style>
  <thead>
    <tr>
      <th>Функция</th>
      <th>Описание</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td><code>**key**</code></td>
      <td>
        Какая сейчас нажата клавиша (например, `"0"`, `"k"`, `"Enter"`, `"Shift"`, `"ArrowUp"`, `"ArrowDown"`, `"ArrowLeft"`, `"ArrowRight"`).
        
        Не работает для пробела.
      </td>
    </tr>
    <tr>
      <td><code>**keyCode**</code></td>
      <td>
        Номер нажатой клавиши на клавиатуре. В отличие от `key`, работает для пробела (код 32) и игнорирует раскладку клавиатуры.
      </td>
    </tr>
  </tbody>
</B.Table>

      <td>

      </td>
    </tr>
    <tr>
      <td><code>**frameCount**</code></td>
      <td>
        Переменная, в которой пишется, сколько кадров уже прошло
      </td>
    </tr>
    <tr>
      <td><code>**fullscreen**(true)</code></td>
      <td>
        Переходит в полноэкранный режим.

        Работает только в функциях вроде `mousePressed()`. Например:

        ```js
        function setup() {...}

        function draw() {...}

        function mousePressed() {
          fullscreen(true);
        }
        ```
      </td>
    </tr>
  </tbody>
</B.Table>
