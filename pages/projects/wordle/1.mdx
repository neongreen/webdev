# Пишем игру Wordle: первый урок

## Что это вообще за игра

В игре <Ann text="word = слово">Wordle</Ann> (Вордл) загадывается слово и надо угадать его за
несколько попыток. Поиграть:

- [wordle.belousov.one](https://wordle.belousov.one/) — Wordle на русском
- [nytimes.com/games/wordle](https://www.nytimes.com/games/wordle/index.html) — Wordle на английском
- [quordle.com](https://www.quordle.com/) — Quordle (четыре слова сразу)
- [octordle.com](https://octordle.com/) — Octordle (восемь слов сразу)

Мы будем делать свою версию этой игры.

## Основы основ: HTML

1.  HTML состоит из _элементов_.
2.  Элементы могут быть внутри других элементов.

Пример:

<Boxes type="plain" center>

## Основы основ: CSS

На языке CSS мы пишем **стили** для элементов. Эти стили говорят, как элемент должен выглядеть.

<Exercise>
  <Step>
    Создайте новый пустой файл <FileName>style.css</FileName>. Туда мы будем писать стили.
  </Step>
  <Step>
    Добавьте в самое начало файла <FileName>index.html</FileName> три строчки:

    ```html index.html mark=1:3
    <head>
      <link rel="stylesheet" href="/style.css">
    </head>

    <body>
      ...
    ```

    Эти строчки говорят, что наша страница должна использовать стили из "style.css".

    В элементе `<head>` пишутся "настройки" страницы — какие стили использовать на странице, какая у нее должна быть иконка, и так далее. `<head>` — это единственный элемент, который должен быть не внутри `<body>`, а снаружи.
  </Step>
  <Step>
    Напишите в <FileName>style.css</FileName>:

    ```css style.css
    body {
      background-color: blue;
    }
    ```

    Страница должна стать пронзительно голубого цвета.
  </Step>
</Exercise>

  </Mistake>
  <Mistake>
    **Селекторы надо писать без угловых скобок.**
    <Boxes type="plain">

## Классы

Если надо поменять стиль не для всех заголовков (или других элементов), а только для некоторых, им надо назначить **класс**. Дальше надо будет написать CSS-правило, которое применяется только к этому классу.

Сейчас мы это сделаем.

<Exercise>
  <Step>
    Сделайте, чтобы на странице было два заголовка. Если уже два — все хорошо.
  </Step>
  <Step>
    Напишите в <FileName>style.css</FileName> правило, которое будет ставить под углом все элементы с классом `angle`:

    ```css style.css
    .angle {
      transform: rotate(-5deg);
    }
    ```

    Пока что ничего не должно случиться, потому что у нас нет элементов с таким классом.
  </Step>
  <Step>
    <Boxes>
      <Box>
        Поставьте класс `angle` одному из заголовков. Должно стать как на картинке.

        ```html index.html mark=1[5:17]
        <h1 class="angle">Заголовок</h1>
        <p>какой-то текст</p>
        <h1>Еще заголовок</h1>
        ```
      </Box>
      <Box>
        <Figure>
          <Mac size="small">
            <div className="reset">
              <h1>Заголовок</h1>
              <p>какой-то текст</p>
              <h1 style={{transform: 'rotate(-5deg)'}}>Еще заголовок</h1>
            </div>
          </Mac>
        </Figure>
      </Box>
    </Boxes>
  </Step>

  <Mistake>
    **Класс нельзя писать снаружи тега.**
    <Boxes>
      <Box>
        🚫 Неправильно:
        ```html index.html mark=1[4]
        <h1> class="angle" Заголовок</h1>
        ```
      </Box>
      <Box>
        ✅ Правильно:
        ```html index.html mark=1[18]
        <h1 class="angle">Заголовок</h1>
        ```
      </Box>
    </Boxes>
  </Mistake>
  <Mistake>
    **Перед `class` должен быть пробел, после — равно.**
    
    `<h1class="angle">` или `<h1 class "angle">` — оба неправильных варианта.
  </Mistake>
  <Mistake>
    **Не забудьте точку в CSS.** А вот в HTML как раз точка не нужна.
    <Boxes>
      <Box>
        🚫 Неправильно:
        ```css style.css
        angle {
          transform: rotate(-5deg);
        }
        ```
      </Box>
      <Box>
        ✅ Правильно:
        ```css style.css mark=1[1]
        .angle {
          transform: rotate(-5deg);
        }
        ```
      </Box>
    </Boxes>
  </Mistake>
</Exercise>
     
Важно:

- `class="red"` надо писать **внутри** открывающего тега — после названия элемента (`h1` или `p` или еще что-то) и до закрывающей угловой скобки `>`.
- В селекторе надо писать точку в начале. Не `red {...}`, а `.red {...}`.

## Бонус: картинки

(Для игры не понадобится.)

Чтобы вставить картинку, нужен элемент `img` (от слова "image" —
изображение).

Скачайте любую картинку из интернета. Зайдите в "Assets" в панели слева
и нажмите "Upload an asset". Потом нажмите на загруженную картинку и
"Copy URL".

Добавьте на странице элемент `img`:

```
<img width="100%" src="...">
```

Вместо `...` вставьте скопированную ссылку на картинку.

У картинок нет закрывающего тега. То есть, `</img>` писать не надо.
