# Пишем игру Wordle: первый урок

## Что это вообще за игра

В игре <Ann text="word = слово">Wordle</Ann> (Вордл) загадывается слово и надо угадать его за
несколько попыток. Поиграть:

- [wordle.belousov.one](https://wordle.belousov.one/) — Wordle на русском
- [nytimes.com/games/wordle](https://www.nytimes.com/games/wordle/index.html) — Wordle на английском
- [quordle.com](https://www.quordle.com/) — Quordle (четыре слова сразу)
- [octordle.com](https://octordle.com/) — Octordle (восемь слов сразу)

Мы будем делать свою версию этой игры.

2.  Элементы могут быть внутри других элементов.

## Основы основ: CSS

На языке CSS мы пишем **стили** для элементов. Эти стили говорят, как элемент должен выглядеть.

Создайте новый файл <FileName>style.css</FileName> и добавьте туда правило, которое покрасит всю страницу в голубой цвет:

```css style.css
body {
  background-color: blue;
}
```

Ничего не случится. Это потому, что мы не подключили файл к нашей странице.

Теперь добавьте в самое начало файла <FileName>index.html</FileName> три строчки:

```html index.html mark=1:3
<head>
  <link rel="stylesheet" href="/style.css">
</head>

<body>
  ...
```

Страница должна стать загадочного голубого цвета.

## Классы

Если надо поменять стиль не для всех заголовков (или других элементов), а только для некоторых, им надо назначить **класс**. Пример:

```html
// mark[5:15]
<h1 class="red">Красный заголовок</h1>

<h1>Обычный заголовок</h1>
```

Теперь можно написать правило, которое применится только к элементам с классом "red":

```css
.red {
  color: red;
}
```

Важно:

- `class="red"` надо писать **внутри** открывающего тега — после названия элемента (`h1` или `p` или еще что-то) и до закрывающей угловой скобки `>`.
- В селекторе надо писать точку в начале. Не `red {...}`, а `.red {...}`.

Напишите в <FileName>style.css</FileName> правила, чтобы в абзаце с классом "big" текст стал большим, а в абзаце с классом "red" — красным.

## Бонус: картинки

(Для игры не понадобится.)

Чтобы вставить картинку, нужен элемент `img` (от слова "image" —
изображение).

Скачайте любую картинку из интернета. Зайдите в "Assets" в панели слева
и нажмите "Upload an asset". Потом нажмите на загруженную картинку и
"Copy URL".

Добавьте на странице элемент `img`:

```
<img width="100%" src="...">
```

Вместо `...` вставьте скопированную ссылку на картинку.

У картинок нет закрывающего тега. То есть, `</img>` писать не надо.
